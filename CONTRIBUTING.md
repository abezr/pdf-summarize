# Contributing to PDF Summary AI

Thank you for your interest! This is a take-home project for COXIT, but feedback and suggestions are welcome.

## Documentation

All documentation is in `/docs`. Start with:
- **[README.md](./README.md)** - Project overview
- **[docs/INDEX.md](./docs/INDEX.md)** - Complete documentation index

## Reporting Issues

If you find bugs or have suggestions:
1. Check existing [issues](https://github.com/abezr/pdf-summarize/issues)
2. Create a new issue with:
   - Clear description
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Environment details (Node version, OS, etc.)

## Questions

For questions about the project:
- Review [Quick Reference](./docs/guides/QUICK-REFERENCE.md)
- Check [LLM documentation](./docs/llm/README.md)
- Open an issue for discussion
